🎨 Kohya LoRA Training

ℹ️ About

Train custom LoRAs using Kohya SS with SD 1.5, SDXL, or Flux.1 models.

  • ⚡ Fast: SD 1.5 trains in ~10 minutes
  • 💾 Low VRAM: Works with 16GB VRAM
  • 🎯 Quality: Style, subject, and concept LoRAs

⚠️ Kohya SS Not Installed

To use the training interface, you need to install Kohya SS:

  1. Download from github.com/bmaltais/kohya_ss
  2. Extract to SwarmUI/../kohya_ss
  3. Run gui.bat
  4. Then refresh this page

After install, use the Kohya web UI at localhost:7860 for full training controls.

🚀 Quick Start

📁 Training Datasets

Place image folders in: training_data/

Tip: Create folders like training_data/my_style/ with 20-50 images each

🎁 Trained LoRAs

LoRAs are automatically saved to: Models/Loras/

📖 Training Workflow

  1. Create a folder in training_data/my_lora
  2. Add 20-50 training images (512px for SD1.5, 768px for SDXL)
  3. Open Kohya Web UI → LoRA tab
  4. Configure:
    • Model: Choose base model (SD 1.5, SDXL, or Flux)
    • Training data: Point to your image folder
    • Output: Models/Loras/my_lora
    • Epochs: 10-20 (usually)
    • Batch size: 1-2 (for 16GB VRAM)
  5. Click Train and wait
  6. LoRA appears here when done ✓

⚙️ Recommended Settings (16GB VRAM)

SD 1.5

  • Batch size: 2-4
  • Resolution: 512x512
  • Training time: ~10 min/10 epochs
  • Use: General styles, subjects

SDXL

  • Batch size: 1-2
  • Resolution: 512x512 or 768x512
  • Training time: ~30 min/10 epochs
  • Use: Higher quality outputs

Flux.1

  • Batch size: 1
  • Resolution: 512x512 (not 1024)
  • Training time: ~45 min/5 epochs
  • Use: Best quality, latest features
  • Note: Use ai-toolkit instead of Kohya for best results
